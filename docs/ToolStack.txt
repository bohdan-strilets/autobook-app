================================================================================
1. Mobile (Frontend)
================================================================================

  React Native + Expo
  Core stack:
    - React Native (Expo SDK)
    - TypeScript
    - Expo Router
    - React Query (TanStack Query)
    - Zustand
    - React Hook Form + Zod
    - NativeWind / Tailwind
    - Reanimated
    - Expo Image
    - Victory / Recharts для графіків
    
  UI
    - Tamagui або
    - NativeWind + headless компоненти
  
  Notifications
    - Expo Notifications
    
================================================================================
2. Backend
================================================================================

  Supabase
  Це ключова технологія проекту.
  Вона дає одразу:
    - PostgreSQL
    - Auth
    - Storage
    - Edge Functions
    - Realtime
    - RLS security
    
  Використовуємо:
    - Database  ->	PostgreSQL
    - Auth      ->	Supabase Auth
    - Storage   ->	Supabase Storage
    - Functions ->	Supabase Edge Functions
    - Security  ->	Row Level Security
